# API Directory

The [`src/api/`](mdc:src/api) directory contains logic for communicating with backend services and external APIs. It includes:

- API endpoint definitions (e.g., `endpoint.ts`, `icp.ts`, `blink.ts`)
- Axios instance configuration (`axiosInstance.ts`)
- Backend service wrappers (`backend.ts`)
- Central API exports (`index.ts`)

Use this directory for all code related to API requests, backend communication, and service integration.
